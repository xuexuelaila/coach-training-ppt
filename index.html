Key Duties

教练核心三件事

1
答疑
船员遇到问题在群内提问，基于自己的经验给建议。费曼学习法的实践 -- 讲给别人听，检验自己是否掌握。
2
作业点评
志愿者每周三整理优质作业，教练在群内做点评。一两句话：好的夸具体好在哪，有改进空间的指方向。
3
陪伴
适当鼓励、推一把，帮船员迈出第一步。"最近是不是卡在哪了？需要帮忙吗？"
09 / 19
Boundaries

三个「不需要」

✗

不需要替船员做

引导船员学会排查方法，学会使用AI解决，而不是直接帮改代码

✗

不需要什么都会

不确定的问题发到教练内部群，大家一起解决，我们是一个团队

✗

不做引流和推广

群内聚焦学习和交流，信任建立后合作机会反而会更多

教练的价值不在于"什么都懂"，而在于"愿意陪着走"

Module 04

航海全流程 · 四阶段

14天航海，四个阶段，每个阶段教练的参与节奏不同

Phase 01
开船前
5月1日起
入群和准备
志愿者拉船员进航海群
每群邀请两位教练入驻

Phase 02
启航第1天
5月7日
船员刚上手
解答方向感问题：从哪开始？
帮忙排查工具操作卡点
每天看2-3次群消息即可
Phase 03
航海中期
5月中旬
答疑和点评
继续群内答疑，不确定的转教练群

Phase 04
冲刺靠岸

作业提醒由领队和志愿者负责
教练保持群内答疑节奏
靠岸后运营做数据汇总和激励

Module 05

答疑三类问题应对

手册里有答案
先给方向性回答，再引导查手册
"在手册第三章有详细步骤，先跟着走一遍，卡在哪一步再发出来"
需要经验判断
分享自己的真实经验
"我之前也遇到过，后来试了XX方法，效果好了很多"
自己也不确定
先回复，转教练内部群
"我帮你问一下" 比沉默好得多
15 / 19
Scenarios

突发情况处理

船员表达不满 / 抱怨

→ 先共情："理解，刚开始确实会有这种感觉"
→ 再引导说具体卡点，帮解决问题
→ 持续激烈的反馈给领队

群内出现广告 / 引流

→ 直接反馈给领队，不需要自己处理

教练临时有事

→ 提前跟领队说，安排储备教练补位，不用有心理负担

16 / 19
Core Rules

答疑核心三原则

1
能答的 → 用经验答
真实的经验比正确的答案更有用
2
不确定的 → 转教练群
背后有教研团队和其他教练协助
3
情绪问题 → 先共情再解决
大部分不满背后是具体的卡点
遇到拿不准的事情，随时找领队和运营团队，大家一起扛
